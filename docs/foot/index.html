Програма курсу

  • День 1: розслаблення гомілки, робота з великогомілковим нервом, м’яка розтяжка
  • День 2: техніки для плантарної фасції та зменшення болю при п’ятковій шпорі
  • День 3: вправи на мобільність гомілкостопу для кращого кроку
  • День 4: відновлення руху великого пальця, профілактика вальгусу
  • День 5: робота з тригерними точками, вправи для суглобів і легке навантаження
  • День 6: зняття напруги при тарзальному синдромі, активація коротких м’язів стопи
Тест

Після етапу 1 проходиш короткий тест — перевіряємо готовність переходити на більшу інтенсивність.
